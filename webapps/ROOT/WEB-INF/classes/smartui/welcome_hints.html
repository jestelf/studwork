Search

  • Select a project and tell Charisma what you're looking for. Smart suggestion will help you to narrow your search down as you type: assigned to me unresolved or fix for version 4.0.x unassigned.
  • Use ! exclamation mark before a word if you want to search issue text: unresolved critical !window.
  • Save search to create an additional navigational means and share it with others.

Navigate

Navigation is easier with keyboard shortcuts:
  • up/down arrow keys to navigate up and down the list,
  • left/right arrow keys to collapse/expand issue details, such as description, comments.
  • Enter opens the selected issue on a separate page,
  • ESC toggles focus between the search field and the issue list.

Edit

  • Change issue attributes right below the issue title by clicking it and selecting a new value.
  • Press F2 or double click on an issue to edit not only attributes but title and description, too.
  • Press Enter or click the title link to navigate to a separate issue.
  • Start typing command while issue is focused to open command window and apply desired command.
